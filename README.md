# 🛒 Super Store Sales Dashboard

This project is an interactive dashboard created using Power BI to analyze sales data of a retail superstore.

---

## 📊 Project Overview

The dashboard provides insights into:

* Sales performance over time
* Profit and loss analysis
* Region-wise sales distribution
* Category and sub-category performance

---

## 🛠 Tools Used

* Power BI
* Excel / CSV
* Power Query (for data cleaning)

---

## 📈 Key Features

* Interactive visuals and filters
* KPI cards (Total Sales, Profit, Quantity)
* Region and category slicers
* Trend analysis using charts

---

## 📁 Files Included

* Super_Store_sales.pbix (Power BI file)
* Screenshot of dashboard (PNG)
